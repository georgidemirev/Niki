# web-backend

## Start project

````
$ npm install (first time or package changes) - install dependencies
$ npm run dev
````

```
	Steps when you want to develop a feature.
	1. Pull dev branch
	2. Create a new branch from dev. Example authentication
	3. Switch to your new branch
	4. Commit
	5. When you are ready, push your changes
	6. Go to **influ-ai-platform** github repository and click "Create a pull request" button
	7. Add title and description
	8. Wait for review approval and merge
```
